# Sistema de Cálculo de Fechas en JavaScript

## Descripción
Este proyecto consiste en una aplicación desarrollada en JavaScript que permite calcular la diferencia entre dos fechas en días, meses y años. Además, incluye una función para formatear fechas en distintos formatos.

## Objetivo
Demostrar el uso del objeto Date en JavaScript aplicado a un caso real como el cálculo de antigüedad de empleados o vigencia de productos.

## Funcionalidades
- Calcular diferencia entre dos fechas
- Mostrar resultados en días, meses y años
- Formatear fechas en formato:
  - DD/MM/AAAA
  - AAAA-MM-DD

## Tecnologías utilizadas
- JavaScript
- Node.js
- Visual Studio Code
